# Casas Fortunae — Как запустить игру на Windows

Полный гайд с нуля. Даже если ты никогда не открывал PowerShell — просто иди по шагам сверху вниз, и через 10 минут вы вдвоём будете играть с телефонов.

Игра работает так: твой компьютер становится маленьким игровым сервером. Оба телефона подключаются к нему по домашнему WiFi и открывают игру в браузере. Никакого интернета игре не нужно (кроме загрузки портретов — но и без них всё работает).

---

## Что понадобится

- Компьютер на Windows (он будет «сервером»).
- Два телефона (или телефон + второй компьютер) — это игроки.
- Все три устройства в **одной WiFi-сети**. Это главное условие.

---

## Шаг 1. Установить Python

Python — это язык, на котором написана игра. Проверим, есть ли он уже.

1. Нажми `Win + X`, выбери **Windows PowerShell** (или **Terminal**).
2. Впиши команду и нажми Enter:

   ```powershell
   python --version
   ```

**Если увидел** что-то вроде `Python 3.11.5` — отлично, переходи к Шагу 2.

**Если увидел ошибку** или открылся магазин Microsoft Store:

1. Зайди на https://www.python.org/downloads/
2. Нажми большую жёлтую кнопку **Download Python 3.x**.
3. Запусти скачанный файл.
4. **ВАЖНО:** на первом экране установщика поставь галочку **«Add python.exe to PATH»** (внизу окна). Без неё команды не заработают.
5. Нажми **Install Now**, дождись конца, закрой окно.
6. **Закрой PowerShell и открой заново** (чтобы он увидел новый Python).
7. Проверь снова: `python --version` — теперь должна показаться версия.

---

## Шаг 2. Положить файлы игры

Проще всего: **распакуй архив `CasasFortunae.zip`** (правый клик → «Извлечь всё»). Внутри уже всё разложено правильно — папки `templates`, `static/music` и файлы на своих местах. Положи распакованную папку, например, на рабочий стол.

Должна получиться такая структура:

   ```
   CasasFortunae\
   ├── server.py
   ├── requirements.txt
   ├── КАК_ЗАПУСТИТЬ.md   (этот файл)
   ├── templates\
   │   └── game.html
   └── static\
       └── music\
           ├── track1_bach.mp3
           ├── track2_haydn.mp3
           └── track3_muti.mp3
   ```

   **Важно:** не растаскивай файлы по отдельности — держи именно эту структуру папок. Если музыка не играет, почти всегда дело в том, что папка `static\music` лежит не рядом с `server.py`. При запуске сервер сам напишет в окне, нашёл он музыку или нет.

---

## Шаг 3. Перейти в папку игры в PowerShell

В том же окне PowerShell нужно «зайти» в папку с игрой командой `cd` (change directory).

Самый простой способ узнать путь: открой папку `CasasFortunae` в Проводнике, кликни один раз на строку адреса вверху — она покажет полный путь. Скопируй его.

Потом в PowerShell впиши `cd `, пробел, и вставь путь в кавычках:

```powershell
cd "C:\Users\ТвоёИмя\Desktop\CasasFortunae"
```

(Подставь свой путь.) Нажми Enter. Строка слева должна смениться на путь к твоей папке.

Проверь, что ты в нужном месте:

```powershell
dir
```

Ты должен увидеть `server.py`, `requirements.txt` и папку `templates`. Если видишь — всё верно.

---

## Шаг 4. Установить нужные библиотеки

Игре нужны три маленькие библиотеки. Установим их одной командой:

```powershell
pip install -r requirements.txt
```

Пойдёт текст, скачаются Flask, flask-sock и qrcode. Это нормально. Дождись, пока снова появится строка ввода.

**Если `pip` ругается «не найдена команда»**, попробуй так:

```powershell
python -m pip install -r requirements.txt
```

---

## Шаг 5. Запустить сервер

Теперь главное:

```powershell
python server.py
```

Если всё хорошо, ты увидишь красивую рамку с адресом и QR-код, примерно так:

```
  ╔═════════════════════════════╗
  ║   http://192.168.1.42:5000  ║
  ╚═════════════════════════════╝

  CASAS FORTUNAE — Дома Фортуны
  ...
  [QR-код]
```

**Запомни этот адрес** (`http://192.168.1.42:5000` — у тебя цифры будут свои). Это адрес твоей игры в домашней сети.

> Окно PowerShell теперь нельзя закрывать — пока оно открыто, сервер работает. Свернуть можно.

---

## Шаг 6. Подключить телефоны

На **каждом** телефоне:

1. Убедись, что телефон в той же WiFi-сети, что и компьютер.
2. Открой браузер (Safari, Chrome — любой).
3. Впиши тот самый адрес: `http://192.168.1.42:5000` (свой).
   - **Ещё проще:** наведи камеру телефона на QR-код в окне PowerShell — он сам предложит открыть ссылку.
4. Откроется заставка **Casas Fortunae**.

Теперь на обоих телефонах:

1. Впиши имя.
2. Оставь вкладку **«Быстрая игра»** (вы оба попадёте в общую комнату).
3. Нажми **«Войти»**.
4. Выбери **Дом** (фракцию) и **портрет**.
5. Нажми **«Готов к битве»**.

Как только оба нажали «Готов» — партия начинается автоматически. Удачи под Колесом Фортуны!

## Музыка, звуки и фоны

**Музыка.** Три классических трека уже лежат в папке `static/music` и подключены к игре. На главном экране внизу — раздел «Музыка двора»: нажми на любой трек, чтобы он заиграл. Музыка не прерывается при переходах между экранами и звучит во время партии. Вверху справа — кнопка управления (пауза/проигрывание) и стрелка переключения на следующий трек. На поле боя она сворачивается в маленький значок, чтобы не мешать.

**Звуки действий.** Каждое событие в бою озвучено: удар по престижу, лечение, набор брони, звон монет, вращение Колеса, шипение яда, перезвон благословения, фанфары победы. Все звуки рождаются прямо в браузере (ничего не скачивается), поэтому работают без интернета. Рядом с кнопкой музыки есть колокольчик 🔔 — нажми, чтобы выключить или включить звуки.

> Браузеры на телефонах не запускают звук сами по себе — это нужно для защиты от навязчивой рекламы. Поэтому музыка и звуки стартуют после первого касания экрана. Это нормально.

Хочешь свои песни? Положи `.mp3` в папку `static/music` и пропиши их в списке `MUSIC_TRACKS` в начале файла `server.py` — там видно по образцу, как добавить название и композитора.

**Фоны.** В меню фоном плавно сменяются шедевры Возрождения. Если у компьютера есть интернет, игра подгружает настоящие картины из коллекции музея Метрополитен (общественное достояние). Если интернета нет — показываются встроенные живописные фоны, так что красиво в любом случае. Во время самой партии фон скрывается за «пергаментом», чтобы поле боя оставалось чистым и читаемым.

## Если что-то не работает

**Телефон пишет «не удаётся открыть страницу» / бесконечная загрузка**

Почти всегда это брандмауэр Windows блокирует подключение. Лечится так:

- Когда ты **первый раз** запустил `python server.py`, Windows мог показать окно **«Разрешить доступ»**. Нужно было нажать **Разрешить** (и желательно отметить «Частные сети»). Если ты случайно нажал «Отмена» — смотри ниже.
- Останови сервер (в PowerShell нажми `Ctrl + C`), запусти заново `python server.py`, и в появившемся окне нажми **Разрешить доступ**.

Если окна не было, открой брандмауэр вручную:

1. `Win + R`, впиши `wf.msc`, Enter.
2. Слева — **Правила для входящих подключений** → справа **Создать правило**.
3. Тип: **Для порта** → Далее.
4. **TCP**, конкретный порт: `5000` → Далее.
5. **Разрешить подключение** → Далее → отметь все три (Домен/Частная/Публичная) → Далее.
6. Имя: `Casas Fortunae` → Готово.

**«python не является внутренней или внешней командой»**

Python не в PATH. Вернись к Шагу 1, переустанови Python с галочкой **«Add python.exe to PATH»**, перезапусти PowerShell.

**Телефон и компьютер точно в одной сети?**

- Бывает, что роутер выдаёт «гостевую» сеть отдельно — подключи телефоны к той же сети, что и компьютер (часто это одно имя WiFi без приставки «Guest»).
- Если у тебя два диапазона (2.4 ГГц и 5 ГГц с разными именами) — подключи все устройства к одному.

**Хочу узнать адрес заново**

Адрес печатается при каждом запуске сервера. Или впиши в PowerShell:

```powershell
ipconfig
```

Ищи строку **IPv4-адрес** (обычно начинается на `192.168...`). Адрес игры — это он плюс `:5000`, например `http://192.168.1.42:5000`.

**Как остановить сервер**

В окне PowerShell нажми `Ctrl + C`. Или просто закрой окно.

## Игра по коду (если «Быстрая игра» не подходит)

Если хотите отдельную комнату (например, играете несколько пар):

1. Один игрок выбирает вкладку **«По коду»** и нажимает **«Создать новую комнату»** — получит код из 4 букв.
2. Второй выбирает **«По коду»**, вписывает этот код и входит.
